# Overview

This project is a JAVA based application to demo a text-based adventure game.
It creates types of characters, simulates combat, and allows the player to make choices.
It currently displays the very basics and could easily scale up with the pieces necessary for differing scenarios.

My goal was to learn the basics of the JAVA language, particularly class inheritance, validating, and decisions.
